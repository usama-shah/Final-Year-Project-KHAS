Khas - Online Mobile Phone Marketplace

Khas is an online marketplace for buying and selling used and new mobile phones. The platform aims to build trust in online shopping by providing a safe and secure environment for users to transact mobile phones. Sellers can post ads with full specifications, and buyers can make purchases through a secure payment system. Khas ensures that payments are held until the phone is verified and delivered, reducing the risk of scams and frauds.

Prerequisites

Before running the Khas project, make sure you have the following installed on your system:

  • PHP (minimum version 8.0)
  • Composer
  • Web server (Apache or Nginx)
  • MySQL or compatible database management system

Installation

To get started with Khas, follow these steps: 


        git clone https://github.com/syed-mushahid/khas.git
        cd khas
        composer install

Configuration

1. Copy the .env.example file to create a .env file. 


        cp .env.example .env

2. Update the .env file with your environment-specific configuration settings, including the database connection details.

You can use and modify the available .env file if you are installing Khas using the zip file provided.

3. Generate a unique application key. 


        php artisan key:generate

Database Setup

1. Create a new database for Khas on your system.

2. Update the database configuration in the .env file with the new database details.

3. Run the database migrations to set up the necessary tables. 


        php artisan migrate

4. Optional: If you have seeders, you can populate the database with sample data. 


        php artisan db:seed

Running the Application

Start the development server to run the application locally. 


        php artisan serve

Access the application in your web browser using the provided URL (usually http://localhost:8000).
